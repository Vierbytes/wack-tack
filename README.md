# Tacky Town
This project represents a store, which sells wacky things, representing strong Tailwind-driven styling for flexibility!

## Tech Stack
- Tailwind CSS
- HTML
- JavaScript
- Bootstrap Icons
- Google Fonts: Bagel Fat One, Funnel Sans

## Features
- Modal CTA popup to prompt user to save on the site via DOM manipulation
- Smooth scrolling for nav links
- Transition animations on button elements and links
- Featured products
- Responsive category filter section
- Custom holiday themes and whimsy imagery with bold colors

## Process
-   Planning and organization via Google Docs
-   Wireframing on Figma
-   Resource gathering
-   Branding and design system determines
-   Creating basic structure, then working in respective parts
-   Using clear and conventional commits, submitting PRs, and reviewing team work
